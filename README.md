# 🏧 ATM Machine Simulator (Python)

A simple **ATM Machine Simulator** developed using Python.  
This project demonstrates the basic working of an ATM system through a command-line interface.

The program allows users to securely access their account using a PIN and perform basic banking operations.

## 📌 Project Overview

The ATM simulator includes a simple workflow:

```

Insert Card
↓
Enter PIN
↓
Select Language
↓
Choose Transaction
↓
Complete Operation

```

## ✨ Features

- 💳 Card insertion simulation
- 🔐 PIN verification
- 🌐 Language selection
- 💰 Withdraw money
- 💵 Deposit money
- 📊 Balance enquiry
- 🔁 Multiple transactions using menu loop
- 🚪 Exit option

## 🛠️ Technologies Used

- Python 3.x
